Manifesto
Mine Free. Exit Fair.

Why this exists

Most wallets are built for traders. This one is built for miners — people who run hardware, pay electricity bills, and contribute real work to securing a network. You deserve a wallet that respects that.

Non-custodial, always

Your keys never leave your device. This wallet generates your seed phrase locally using BIP-39 — the same standard used across the entire crypto industry. No server ever sees your private key.

The exit fee

When you send PRL out of this wallet, a flat 1 PRL community fee is deducted and routed on-chain to the MWU community pool. You can see exactly where it goes before every transaction — transparent, fixed, fair.

Open source

The code for this wallet is public on GitHub. Read it, audit it, fork it. Trust is built through transparency, not promises.

Live balance & price

Your balance is fetched live from the pearlchain.live API. PRL/USD price updates automatically. No accounts, no tracking.
